# Changelog

バージョン規則: [Semantic Versioning](https://semver.org/)
- `0.x.0` 機能追加・画面変更
- `0.x.y` バグ修正のみ
- `1.0.0` 基本機能が固まって本番運用レベル

---

## 0.2.0 - 2026-04-10

### 追加
- 授業記録の編集機能（日付・ステップ・単元・点数・コメントを変更可能）
- 授業記録の削除機能（答案画像の Storage 削除を含む）
- 編集・削除権限制御（入力者本人または admin のみ操作可能）

### 変更
- `calcCompletionType` / `calcScoreSource` を `lib/lesson-record-utils.ts` に共通化

---

## 0.1.0 - 2026-03-18

### 追加
- 初期リリース
- 生徒管理（登録・編集・科目・コンテンツグループ選択）
- 授業記録登録（フォト優先3ステップ UI + Gemini AI による答案解析）
- 単元別進捗表示（生徒詳細ページ・カレンダー・学習ペース）
- 保護者レポート出力（印刷対応 HTML レポート）
- 理科コンテンツ追加（小学4〜6年・中1理科・入門1年）
- AI 解析の精度改善（英文法 入門・理科1年 誤分類修正など）
- リファクタリング（hooks/lib への分割）
